# Massimiliano Aresu

A simple web-based platformer game portfolio built using React, Tailwind CSS, and the Kaplay game engine.

## Installation

To run the game locally, you will need to have Node.js and npm installed on your machine. Then, run the following commands in your terminal:

- Install dependencies:
  `npm install`
- Run the project locally:
  `npm run dev`
